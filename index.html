Cheat Sheet de CSS

Pseudo elementos
1 / 15

¿Qué es un Pseudo-elemento?

Es una palabra clave añadida a un selector que permite estilizar una parte específica del elemento seleccionado.

/* Diferencia clave: */
p:hover { ... }    /* Pseudo-clase (Estado) - Un punto */
p::first-line { ... } /* Pseudo-elemento (Parte) - Dos puntos */

/* Nota: En CSS antiguo se usaba un solo punto, 
   pero la sintaxis moderna :: es la recomendada. */
2 / 15

Insertando Contenido: ::before

Crea un elemento virtual que es el primer hijo del elemento seleccionado. Requiere la propiedad content.

h1::before {
  content: "★ ";
  color: gold;
}

/* Resultado visual: ★ Título Principal */
3 / 15

Insertando Contenido: ::after

Crea un elemento virtual que es el último hijo del elemento seleccionado.

a::after {
  content: " ↗";
  font-size: 0.8em;
}

/* Resultado visual: Leer más ↗ */
4 / 15

La Propiedad `content`

Es obligatoria para ::before y ::after. Si no la pones, el pseudo-elemento no se genera.

div::before {
  content: ""; /* Vacío (para formas o bloques) */
  content: "Hola"; /* Texto */
  content: url(icono.svg); /* Imagen */
  content: attr(href); /* Valor de un atributo HTML */
}
5 / 15

Uso Común: Figuras Geométricas

Puedes crear círculos, cuadrados o líneas decorativas usando content: "".

.btn::after {
  content: "";
  display: block;
  width: 10px;
  height: 10px;
  background: red;
  border-radius: 50%;
}
6 / 15

Tipografía: ::first-letter

Selecciona la primera letra de la primera línea de un bloque. Ideal para efectos de "Letra Capital" (Drop Cap).

p::first-letter {
  font-size: 3em;
  font-weight: bold;
  color: #4fc3f7;
  float: left;
  line-height: 1;
}
7 / 15

Tipografía: ::first-line

Selecciona la primera línea de un bloque de texto. Se adapta automáticamente si cambias el ancho de la ventana.

p.intro::first-line {
  font-variant: small-caps;
  font-weight: bold;
}

/* Solo acepta propiedades de fuente, color y fondo */
8 / 15

Interacción: ::selection

Controla el estilo del texto cuando el usuario lo selecciona con el ratón.

::selection {
  background-color: #4fc3f7;
  color: black;
}

/* ¡Haz que la selección combine con tu marca! */
9 / 15

Formularios: ::placeholder

Estiliza el texto de ayuda dentro de los inputs.

input::placeholder {
  color: #888;
  font-style: italic;
  opacity: 0.7;
}
10 / 15

Listas: ::marker

Permite cambiar el color o fuente de los puntos (balas) o números de una lista.

li::marker {
  color: red;
  font-size: 1.2em;
}

/* Mucho más fácil que los trucos antiguos con ::before */
11 / 15

Modales: ::backdrop

Estiliza el fondo que cubre la página cuando se abre un <dialog> nativo.

dialog::backdrop {
  background-color: rgba(0, 0, 0, 0.8);
  backdrop-filter: blur(5px);
}
12 / 15

Archivos: ::file-selector-button

El infame botón de "Examinar..." o "Seleccionar archivo" ahora se puede estilizar.

input[type="file"]::file-selector-button {
  background: #4fc3f7;
  border: none;
  padding: 10px;
  border-radius: 5px;
}
13 / 15

Posicionamiento con ::before/::after

El combo más poderoso: Posición absoluta dentro de un padre relativo.

.padre { position: relative; }

.padre::before {
  content: "";
  position: absolute;
  top: 0; left: 0;
  /* Esto crea capas decorativas perfectas (overlays) */
}
14 / 15

Resumen

Diferencias clave para recordar.

1. Usa :: (doble) para elementos, : (simple) para estados.
2. ::before y ::after SIEMPRE necesitan 'content'.
3. Los pseudo-elementos NO están en el HTML (DOM).
4. No los uses para información crítica (lectores de pantalla
   a veces los ignoran).
5. Úsalos para decoración, iconos y efectos visuales.
